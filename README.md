# Financial RAG Knowledge Assistant

An AI-powered Retrieval-Augmented Generation (RAG) application that answers questions from financial reports using semantic search and LLMs.

## Features

- Upload financial PDF documents
- Automatic document chunking
- Vector embeddings
- Semantic search with FAISS
- LLM-powered question answering
- Streamlit web interface

## Tech Stack

- Python
- LangChain
- OpenAI API (or Gemini)
- FAISS
- Streamlit
